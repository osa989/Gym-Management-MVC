# Gym Management System

A Razor Pages / ASP.NET Core MVC gym management application.  
This solution provides member management, health records, and basic CRUD operations. The web UI project is `GymManagementPL`. Target framework: .NET 9 (C# 13).

## Key features
- Member CRUD (personal + address + health data)
- Server- and client-side validation using DataAnnotations
- Bootstrap-based responsive UI (Views in `GymManagementPL/Views/Member`)
- Clean layered solution with BLL and DAL projects

## Tech stack
- .NET 9
- C# 13
- ASP.NET Core (Razor Pages / MVC)
- Entity Framework Core (if present in DAL)
- Bootstrap for UI

## Prerequisites
- .NET 9 SDK installed
- __Visual Studio__ 2022 (latest updates) or CLI tools
- Optional: SQL Server / local DB if the project uses a database
